# music_player_web

A simple web-based music player built using HTML, CSS, and JavaScript.

## Overview

This project demonstrates a basic music player interface with audio playback functionality.  
It focuses on clean structure, readable code, and straightforward UI behavior.

- React.js
- Vite
- JavaScript (ES6+)
- Redux Toolkit
- React Context API
- Tailwind CSS
- HTML5
- Node.js
- npm
